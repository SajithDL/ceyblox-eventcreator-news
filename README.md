# CeyBlox Event Creator News Feed

This repository stores the public dashboard news, update banners, and patch notes used by **CeyBlox Event Creator**.
The FiveM resource can read this repository through raw GitHub URLs and show update cards inside the in-game dashboard.

## Purpose

This repository is used for:

- Dashboard update banners
- Version update notices
- Patch notes
- Addon release announcements
- Important support or documentation notices

This repository does **not** contain the CeyBlox Event Creator source code.
